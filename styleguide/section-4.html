4 #permalink Buttons

Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

4.1 #permalink Basic button states

Buttons have :hover, :focus, and :active states

Default styling
.btn-icon-left
Button with icon on the left
.btn-icon-right
Button with icon on the right
.disabled
Disabled button 
<div class="form-actions">
	<button type="submit" class="btn [modifier class]">.btn</button>
</div>

4.2 #permalink Sizing

Default styling
.btn-large
Large button.
.btn-small
Small button. 
<div class="form-actions">
	<button type="submit" class="btn [modifier class]">.btn</button>
	<button type="submit" class="btn btn-icon-left [modifier class]">Icon left</button>
	<button type="submit" class="btn btn-icon-right [modifier class]">Icon right</button>
</div>
